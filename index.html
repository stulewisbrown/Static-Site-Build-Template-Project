OK, so what is this?

This is a static blog style website built on a load of awesome, up-to-date principles of web developement.

Here's a list:

  • Totally responsive
  • Totally browser compatible
  • Totally modular (see Kitchen Sink page for examples of components)
  • Totally all about BEM principles
  • Totally HTML5 and CSS3 compliant (with fallbacks of course)

There's a load of great tech behind the build of the site as well

Here's another list:

  • Inuit CSS framework
  • Jekyll static site generator
  • Gulp build system - used to implement awesome stuff like:
    • Browser Sync
    • Jekyll build/rebuild commands
    • Autoprefixing, concatenation, minification...
    • To see the rest, check out the github repo for the build files
